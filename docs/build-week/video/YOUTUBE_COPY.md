## Description

GuardianNode is a local-first family safety monitor. Guardian Review is its
Build Week 2026 extension: an optional, parent-approved GPT-5.6 second opinion
that turns a locally detected incident into uncertainty-aware conversation
guidance.

This video uses GuardianNode's functional synthetic demo and deterministic mock
Guardian Review mode. It contains no real child or family data and makes no
live OpenAI request. Configured live mode uses the server-side OpenAI Responses
API with GPT-5.6, strict Structured Outputs, no tools, and `store:false` after
the parent previews and approves the exact minimized context.

Repository: https://github.com/the-vibe-dev/guardiannode

Build Week baseline: https://github.com/the-vibe-dev/guardiannode/tree/pre-build-week-2026

Build Week comparison: https://github.com/the-vibe-dev/guardiannode/compare/pre-build-week-2026...main

GuardianNode is alpha software. Guardian Review is advisory and can be wrong.
It does not determine truth, diagnose conditions, make punishment decisions, or
replace emergency services, parenting, or qualified professional help.

Pre-existing work disclosure: the agent, backend, local detection, encrypted
evidence, dashboard, authentication, and installers existed before Build Week.
The owner describes portions of the earlier visual web UI as Claude-assisted
and the broader platform as Codex-built. Guardian Review was the Build Week
extension developed with Codex and GPT-5.6.
