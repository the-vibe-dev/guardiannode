# Support Development

GuardianNode is AGPL-3.0 open-source software. It does not require a cloud
account or subscription, and donations are optional.

Donations help fund practical release work such as test hardware, Windows code
signing, installer testing, documentation, security review, and future platform
support.

## Expectations

Donations do not buy access to child data, telemetry, hidden features, or a
priority support queue. Please use GitHub Issues or Discussions for
non-sensitive project questions, and do not upload child screenshots, private
messages, evidence exports, or personal logs to public issues.
